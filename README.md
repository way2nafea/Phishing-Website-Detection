# NEXUS: Phishing Website Detection and Analysis System

**Abstract:** Phishing remains a significant cyber threat, involving the use of fraudulent websites to steal sensitive user information. This project, titled NEXUS, proposes a machine learning-based approach to detect such malicious URLs. By analyzing lexical features and structural patterns of websites in real-time, the system provides an accurate risk assessment to enhance cybersecurity defense mechanisms.

### Technology Stack
- **Programming Language:** Python
- **Web Framework:** Flask
- **Machine Learning:** Scikit-learn
- **Frontend:** HTML, CSS, JavaScript

### Key Features
- Multi-layer analysis of URL lexical features.
- Real-time DNS and SSL certificate validation.
- Integration with third-party threat intelligence APIs.
- Dynamic risk scoring and forensic explanation engine.
- Persistent monitoring and scan history.

### Deployment Instructions
1. Install the required dependencies using the command: `pip install -r requirements.txt`.
2. Initialize and execute the application by running: `python app.py`.
3. Open a web browser and navigate to `http://localhost:10000` to access the interface.
